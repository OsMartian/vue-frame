# vue-frame

> A Vue.js2.0 frame project include axios、vue-router、vuex...

# 项目结构

```
weex-frame
│
├── build (webpack 打包配置)
│       
├── config (环境配置)
│
├── test (测试模块)
│
└── src (主业务)
    │
    ├── api (api模块)
    ├── assets (api模块)
    ├── components（组件模块） 
    ├── pages （界面）
    ├── router (路由)
    ├── store （vuex）
    └── utils (工具模块)   

```

# 项目启动

```
// install depend
npm install
// start
npm run dev
```

* 打开 [http://localhost:8080/](http://localhost:8080/)
